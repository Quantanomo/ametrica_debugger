# Ametrica Debugger

Простое Chrome‑расширение для просмотра событий Альфаметрики (Snowplow) прямо в браузере.
Показывает последние события, их ключевые параметры и custom dimensions.

## Что делает
- Перехватывает запросы к `metrics.alfabank.ru`
- Декодирует `cx` (Snowplow contexts) и показывает custom dimensions
- Хранит последние 200 событий локально (TTL 1 час)
- Позволяет фильтровать, очищать и экспортировать события

## Установка
1. Скачайте проект как ZIP.
2. Распакуйте ZIP в любую папку на компьютере.
3. Откройте в Chrome страницу `chrome://extensions`.
4. Включите **Developer mode** (переключатель в правом верхнем углу).
5. Нажмите **Load unpacked**.
6. Выберите папку с расширением (где лежит `manifest.json`).
7. Убедитесь, что расширение появилось в списке и включено.

## Как пользоваться
1. Нажмите на иконку расширения в панели браузера.
2. В popup нажмите **Start logging**.
3. Откройте сайт `alfabank.ru` и выполните нужные действия.
4. Вернитесь в popup — появится список событий.
5. Используйте:
   - поле фильтра (точное совпадение)
   - чекбокс «Скрывать pp (page ping)»
   - кнопки **Start logging** / **Stop logging** для управления записью
   - кнопку **Clear** для очистки
   - кнопку **Export JSON** для выгрузки

## Примечания
- `pp` (page ping) приходит регулярно — это нормально.
- Если `cx` повреждён или слишком большой (больше 100KB), будет показана ошибка декодирования.
- Данные хранятся локально и автоматически очищаются через 1 час.

## Удаление
Откройте `chrome://extensions` → найдите расширение → нажмите **Remove**.
